### 1. 标题

使用 `#` 表示标题，`#` 的数量决定标题级别。

### 2. 文本样式

加粗：**包裹**
斜体：*包裹*
划线：~~包裹~~



### 3. 列表

- 无序列表：用 `-`、`+` 或 `*` 表示
- 有序列表：用数字加点 `1.` 表示
### 4. 链接和图片

- 链接：`[链接文本](链接地址)`
- 图片：`![图片描述](图片链接)`
### 5. 引用

使用 `>` 表示引用，可以嵌套。
### 6. 分割线

使用 `---`、`***` 或 `___` 创建分割线。

### 7. 代码

- 行内代码：使用反引号 ` 包裹代码
- 块级代码：使用三个反引号 ``` 包裹
### 8. 表格

使用 `|` 和 `-` 创建表格。
### 9. 任务列表

使用 `- [ ]` 表示未完成，`- [x]` 表示完成。
### 10. 转义字符

使用 `\` 转义 Markdown 特殊字符。
